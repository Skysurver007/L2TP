
Instal L2TP
✅ Dengan curl:
```bash <(curl -s https://raw.githubusercontent.com/Skysurver007/L2TP/refs/heads/main/l2tp.sh)```

✅ Atau dengan wget:
```bash <(wget -qO- https://raw.githubusercontent.com/Skysurver007/L2TP/refs/heads/main/l2tp.sh)```
